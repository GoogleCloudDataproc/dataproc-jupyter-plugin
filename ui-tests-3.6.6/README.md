# Integration Testing (JupyterLab 3.6.x)

This folder contains the integration tests of the extension run against **JupyterLab 3.6.x**
(Galata `4.5.6`). The tests, commands, and workflow are identical to the JupyterLab 4.x suite.

**See [`../ui-tests/README.md`](../ui-tests/README.md) for the full instructions** on running,
updating snapshots, creating, and debugging the tests — substituting this `ui-tests-3.6.6`
directory wherever `ui-tests` is referenced.
